üöÄ Create Infrastructure

./scripts/create_infra.sh
  1. Stop EMR if running (for Terraform updates)
  2. Terraform apply ‚Äî VPC, MSK, EMR, ECS, S3 Tables
  3. Configure S3 Tables ‚Äî Intelligent-Tiering
  4. Build & push Docker ‚Äî Producer image to ECR
  5. Upload Spark jobs ‚Äî Including DQ module (dq.zip)
  6. Start ECS producer ‚Äî desired_count = 1
  7. Start Bronze streaming ‚Äî EMR Serverless job
  8. Schedule batch pipeline ‚Äî EventBridge (15 min)

üõë Destroy Infrastructure

./scripts/destroy_all.sh

13-step teardown:

  • Cancel running EMR jobs
  • Stop EMR application
  • Stop ECS service
  • Delete EventBridge schedules
  • Empty S3 buckets
  • Delete S3 Tables (async)
  • Terraform destroy
  • Final verification
Idempotent: Safe to run multiple times
